Aniro

Aniro is the library that allow you to animate your page effortlessly. Just install the library and add the aniro tag to the body – your page will be brought to life instanlty.

Simple

Just plug the library in – the under-hood magic will do the rest.

Easy

You don't need to write the animation code anymore.

Declarative

Configuration is mandatory and really easy though. Just visit the docs page for instructions.
